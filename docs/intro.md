# Coffee Repair Wiki

A practical knowledge base for diagnosing and repairing espresso machines — built from real repair work, starting with Breville and expanding to any brand.

## How to use this wiki

**New to machine repair?** Start with the [Foundation](/foundation/machine-taxonomy) section — brand-agnostic concepts that transfer to every machine you'll ever work on.

**Looking for a specific model?** Jump straight to [Model Dossiers](/models/BES870) — each page covers component specs, diagnostic trees, common failures, and part numbers.

| Level | Skills | Example Jobs |

| **Advanced** | PID tuning, boiler rework, PCB component-level diagnosis, E61 group service | Dual boiler calibration, commercial overhaul |
